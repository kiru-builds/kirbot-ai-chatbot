## 💬 KirBot — AI Chatbot

*By Kirthika | AI & Data Science Student*

## 📌 About
KirBot is an intelligent conversational AI Chatbot built with 
Python and Tkinter. It features a beautiful dark theme GUI and 
can answer questions about AI, ML, Python and career advice!

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- Threading
- DateTime

## 📊 Features
- Beautiful dark theme chat interface
- Real time responses with typing delay
- Knows about AI, ML, Python, Data Science
- Career advice for internships abroad
- Quick action buttons
- Tells jokes!

## 🚀 How to Run
pip install tkinter
python chatbot.py

## 💬 Sample Questions
- What is AI?
- What is Machine Learning?
- Tell me a joke
- How to get internship abroad?
- What is Python?
- Who made you?
